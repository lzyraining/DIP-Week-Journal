Pseudocolor Image Processing

Intensity Slicing

Written by Zhuoyu Li on Mar. 24, 2015.

Sometimes we can only get graylevel image due to the image detector. Here we can use pseudocolor image processing method get more explicit information. An easy way to do pseudocolor processing is intensity slicing. The intensities above the slice are assigned to one color, intensities below the slice are assigned to another color. Here we start with lena graylevel image. And we assign eight color "blak blue red magenta green cyan yellow white". The following two images are input lena graylevel image and output pseudocolor image.

From the pseudocoloar lena image we can directly get the intensity distribution information. In my code, I didn't assign the intensities which are on the intensity slice plane randomly to two color. As a result, the interfaces between two adjacent color are less smooth.

Matlab Code:

clear;clc;
A=imread('lena.jpg');
[row col]=size(A);
B=zeros(row,col);
B=cat(3,B,B,B);

for i=1:1:row
    for j=1:1:col
        if (A(i,j)>=0)&&(A(i,j)<32)
            B(i,j,1)=0;
            B(i,j,2)=0;
            B(i,j,3)=0;
        elseif (A(i,j)>=32)&&(A(i,j)<32*2)
                 B(i,j,1)=0;
                 B(i,j,2)=0;
                 B(i,j,3)=255;
        elseif (A(i,j)>=32*2)&&(A(i,j)<32*3)
                 B(i,j,1)=0;
                 B(i,j,2)=255;
                 B(i,j,3)=0;
        elseif (A(i,j)>=32*3)&&(A(i,j)<32*4)
                 B(i,j,1)=0;
                 B(i,j,2)=0;
                 B(i,j,3)=255;
        elseif (A(i,j)>=32*4)&&(A(i,j)<32*5)
                 B(i,j,1)=255;
                 B(i,j,2)=0;
                 B(i,j,3)=0;
        elseif (A(i,j)>=32*5)&&(A(i,j)<32*6)
                 B(i,j,1)=255;
                 B(i,j,2)=0;
                 B(i,j,3)=255;
        elseif (A(i,j)>=32*6)&&(A(i,j)<32*7)
                 B(i,j,1)=255;
                 B(i,j,2)=255;
                 B(i,j,3)=0;
        elseif (A(i,j)>=32*7)&&(A(i,j)<32*8)
                 B(i,j,1)=255;
                 B(i,j,2)=255;
                 B(i,j,3)=255;
        end
    end
end
imwrite(B,'pseudolena.jpg');

Intensity to color transformtion

Written by Zhuoyu Li on Mar. 24, 2015.

Another way to transform graylevel image to pseudocolor image is using three independent tranforming functions to get RGB information of the new image. Following image illustrates the conception of the intensity to color transformation. (Refer to Gonzale's book Chapt.6)

Here's the lena graylevel image and output result. And we compare the output resutl with the third image which using matlab command "ShowImage(ImgGray,’Pseudo Image’,jetcolormap(256))".

The differece between output image and the third image is using different transformation functions (colormap). We can see that different color region in two images are same, which indicate the differnt intensties in graylevel image. Then we show a moon image dealing with pseudocolor processing.

By tranforming the intensity to pseudocolor informtaion, we can dircetly konw the geomorphology knowledge of the moon at a glance.

Matlab Code:

clear;clc;
A=imread('lena.jpg');
[row col]=size(A);
A=double(A);
B(:,:,1)=sin(2*pi*A(:,:)/256+pi/2);
B(:,:,2)=sin(2*pi*A(:,:)/256+pi/3);
B(:,:,3)=sin(2*pi*A(:,:)/256+pi/4);

B=abs(B)*256;
B=uint8(B);
imwrite(B,'intensitylen.jpg');
